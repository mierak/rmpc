Currently only linux is officially supported. No binaries are currently provided, so you will need to build from source.
To build you will need to have [rust toolchain](https://www.rust-lang.org/tools/install) installed. More installation
methods will be provided in the future.

## Steps

<Steps>

1. Install rmpc with one of the following methods
    - By downloading a precompiled binary from github [releases page](https://github.com/mierak/rmpc/releases/latest/).
      Then untar the archive, set execute persmission if needed and run rmpc.
    - Using cargo
    ```bash frame=none showLineNumbers=false
    cargo install rmpc --locked
    ```
    - Using [cargo-binstall](https://github.com/cargo-bins/cargo-binstall)
    ```bash frame=none showLineNumbers=false
    cargo binstall rmpc
    ```
    - Install the latest git version
    ```bash frame=none showLineNumbers=false
    cargo install --git https://github.com/mierak/rmpc --locked
    ```
2. Setup your config file

    <LinkCard
        title="Configuring rmpc"
        description="Generate sample config file for rmpc and customize it to your liking."
        href="/rmpc/configuration/"
    />

3. Run rmpc
    ```bash frame=none showLineNumbers=false
    rmpc
    ```

</Steps>

:::note
Minimum supported MPD version is 0.23.5. Other versions may work, but you are on your own.
:::
