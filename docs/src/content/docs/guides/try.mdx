## Nix

[`rmpc`](https://search.nixos.org/packages?channel=unstable&query=rmpc) is available in the `unstable` channel. If you have [Nix](https://nixos.org/) installed, you can start it like so:

```bash frame=none showLineNumbers=false
nix run nixpkgs#rmpc
```
